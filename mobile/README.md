# Addis Beakal Mobile

Expo Router app for the Addis Beakal mobile experience.

## What it includes

- Premium mobile-first home, explore, business detail, saved, profile, login, and signup screens
- Reused Addis categories, neighborhoods, business records, and published reviews from the repo
- Local saved-state and lightweight mobile profile state powered by AsyncStorage
- Shared-data generation via `npm run mobile:sync-data` from the repo root

## Local commands

```bash
cd mobile
npm install
npm run lint
npm run typecheck
npx expo start
```
